# FocusGuard: AI-Powered Anti-Distraction Tool

FocusGuard is a powerful browser extension designed to help you stay focused by monitoring your web activity and blocking distracting content based on your current goals. It uses advanced AI (Gemini) for context-aware filtering, while providing core features that work completely offline.

## 🚀 Key Features

- **🎯 Context-Aware Blocking**: Uses Gemini AI to analyze if a website or specific content (like a YouTube video or Reddit thread) is relevant to your current goal.
- **👁️ Tunnel Vision Mode**: Keeps you focused on your task by dimming non-essential parts of the screen.
- **🔇 Silencer**: Minimize distractions on popular platforms like YouTube, Reddit, and LinkedIn by hiding recommendations, comments, and sidebars.
- **🌑 Grayscale Mode**: Reduce the visual appeal of distracting sites by turning them grayscale.
- **📊 Focus Monitoring**: Track your focus sessions and stay accountable.
- **🛡️ Privacy First**: Core features work 100% offline. AI features are optional and require a backend setup.

## 🛠️ Technology Stack

- **Extension**: HTML, CSS, JavaScript (Manifest V3)
- **Backend**: Node.js, Express, MongoDB, Supabase
- **AI**: Google Gemini API
- **Payment**: Stripe Integration

## 📥 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/cupcakesandcodes/focusguard.git
   ```
2. Load the extension in Chrome:
   - Open `chrome://extensions/`.
   - Enable "Developer mode".
   - Click "Load unpacked" and select the root directory of this project.
